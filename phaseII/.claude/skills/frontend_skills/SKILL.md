# 🧩 Skill Prompt — Frontend Skill (Next.js App Router + Responsive UI)

This skill is focused on building responsive, accessible, and maintainable frontend UI using **Next.js App Router**.

This skill should implement modern UI screens and reusable components without breaking application features.

## Responsibilities
- Build UI using Next.js App Router (`app/` directory structure)
- Create reusable components with clean props and structure
- Implement layouts, nested routes, and dynamic routes correctly
- Build responsive pages using Tailwind CSS (mobile-first approach)
- Integrate frontend UI with backend APIs (FastAPI / REST endpoints)
- Handle server components vs client components correctly
- Implement forms with good structure for validation and error handling
- Manage UI state (loading, error, empty, success)
- Ensure accessibility and semantic HTML best practices
- Optimize rendering and avoid unnecessary re-renders
- Maintain consistent UI styling and design system patterns

## Guidelines
- Follow Next.js best practices for App Router
- Prefer reusable UI components over duplicated code
- Keep code clean, readable, and scalable
- Always include loading and error states for async pages
- Avoid mixing backend logic into UI components
- Use clean layout patterns (grid, flex, spacing, typography)
- Keep responsive design consistent across devices

## When to Use This Skill
Use this skill whenever:
- Creating or updating UI pages in Next.js
- Building responsive components using Tailwind
- Fixing routing, layout, or styling issues
- Integrating frontend with backend APIs
- Improving UI consistency and maintainability
