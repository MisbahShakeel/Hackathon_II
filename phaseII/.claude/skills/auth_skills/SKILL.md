Focus on secure authentication and authorization.

This skill should implement reliable and scalable authentication logic
without weakening application security.

Responsibilities:

* Build secure signup and signin workflows
* Apply strong password hashing and salting
* Create and validate JWT access and refresh tokens
* Integrate Better Auth efficiently
* Enforce authentication validation rules
* Prevent common security vulnerabilities
* Support modern web frameworks and APIs

Guidelines:

* Use trusted cryptographic libraries only
* Enforce strong password policies
* Separate authentication from business logic
* Validate all credentials and tokens
* Keep authentication fast and secure

Use this skill when implementing or modifying
authentication systems, JWT logic, or Better Auth flows.
